# 字幕匹配

MoviePilot V2 插件，提供手动上传字幕并匹配改名的页面。

## v0.1.75 更新

- 启用 MoviePilot 插件商店版本历史安装能力，字幕匹配会通过 GitHub Release 工作流生成规范版本包。
- 当前版本对应 Release tag `SubtitleManualUpload_v0.1.75`，安装资产名为 `subtitlemanualupload_v0.1.75.zip`。
- 如升级后遇到问题，请在插件商店的版本历史弹窗中选择稳定版 `v0.1.70` 回退安装，确认稳定后再升级新版。

## v0.1.74 更新

- 主入口进一步瘦身为 MoviePilot 插件壳，剩余运行时 helper、服务访问器和服务代理迁移到独立模块。
- 完成真实 Chrome 浏览器验收，覆盖资源选择、外挂字幕、OpenSubtitles 在线预览、AI 状态和响应式页面。

## v0.1.73 更新

- 移除 `compat.py`、`compat_core.py` 和 `compat_services.py` 兼容层，运行时服务装配改为显式模块。
- 清理兼容库存脚本和旧私有入口测试，后续维护以 API、服务对象和纯函数边界为准。

## v0.1.72 更新

- 拆分插件 API 编排到 `api/` 领域模块，主入口继续收缩为生命周期、配置初始化、事件入口和服务装配。
- 保持原有 API 端点、配置字段和页面操作路径兼容，降低后续继续清理兼容层的风险。

## v0.1.71 更新

- 拆分字幕匹配主入口职责，提取兼容层、配置、上传会话、目标解析、字幕写入和匹配历史等模块，降低后续维护成本。
- 保持现有 API、配置项和页面入口兼容，升级后功能行为不做额外变化。

## v0.1.70 更新

- 放宽 `rarfile`、`numpy`、`pysubs2`、`webrtcvad-wheels` 依赖版本约束，兼容 MoviePilot 运行环境中已安装的更高版本依赖。

## v0.1.69 更新

- 刷新媒体库资源清单改为后台执行，避免资源较多时接口长时间阻塞导致页面超时。
- 新增“信任整理历史路径”兼容模式，CD2、网盘挂载、SMB 等慢路径可开启，刷新资源清单时不逐条访问文件。
- 网盘挂载用户暂不保证完整可用性：开启兼容模式后会减少批量读取媒体库硬盘/网盘，但已有外挂字幕、内置字幕等状态可能无法准确探测。
- 新增 ChineseSubFinder 字幕匹配核心迁移开发方案文档，后续考虑把搜索匹配核心迁回 CSF 后端。

## v0.1.61 更新

- 优化匹配历史里单字幕“调轴 / 删除”按钮的可见性，增加统一背景框、收紧间距并保持右对齐。
- 媒体详情内的匹配历史同步支持单字幕调轴和删除，排版与总匹配历史保持一致。

## v0.1.60 更新

- 修复 SubHD 下载按钮 sid 解析过窄的问题，支持 `/down/` 后非数字短码，例如 `bqsyNZ`。
- 保留 API 返回空地址时解析 `/down` 页面下载链接的兜底流程，提升 SubHD 下载成功率。

## v0.1.59 更新

- 匹配历史改为长期缓存并加入本地视频文件系统签名校验，已删除媒体会自动剔除，同路径重新入库不会被短期去重误拦。
- 入库自动字幕处理改为后台队列，页面可查看等待、处理中、完成、跳过或失败状态，并按 ASSRT / OpenSubtitles 每源 5 次/分钟限速。
- 批量剧集入库优先聚合同季任务搜索整季字幕包，命中后缓存整季包并为后续集数复用，未命中再按单集兜底搜索。
- 匹配历史新增历史外挂字幕智能调轴入口，支持单字幕、选中多集和整季/全部批量提交，状态复用智能调轴任务卡。
- Zimuku 默认地址迁移到 `https://zmk.pw`；旧默认配置会自动迁移，用户手动配置的其他地址不覆盖。
- SubHD 下载增强 API 成功但无直链时的兜底解析，支持 API HTML/url、跳转页和 `/down` 页面下载链接。

## v0.1.58 更新

- 重构在线字幕搜索为 provider 架构，保留原在线搜索、指定来源搜索、下载预览等接口兼容。
- 新增 SubHD、Zimuku 在线源，支持标题关键词搜索、候选解析、下载和验证码识别兜底；默认不加入无人值守自动入库，避免站点波动影响稳定性。
- 修复自动入库字幕处理：无人值守路径仅使用 ASSRT 和 OpenSubtitles，并允许多候选时选择排序最佳的高置信结果。
- OpenSubtitles 优先使用 TMDB ID 搜索，失败后回到原有英文标题搜索策略；ASSRT 继续按标题关键词搜索。
- 在线英文字幕提交 AI 翻译时不再进入下载匹配预览，改为后台下载并直接提交 `AI字幕生成(联动版)`，页面跳转到 AI 状态卡。
- 优化匹配历史和 AI 调轴状态展示，减少重复加载；首页封面改用缩略图懒加载，降低首屏压力。
- 修复字幕语言后缀识别：从字幕扩展名前向前匹配，避免 `KORSUB...eng.srt` 被误判为韩语。

当前能力：

- 从 MoviePilot 本地整理记录中搜索电影或剧集，只展示本地真实存在的视频资源。
- 第一层搜索/选择本地电影或剧集；选中剧集后进入每集列表。
- 剧集支持横向季度卡片，可切换单季或“全部季”。
- 每集列表支持全选/反选、单集上传、锁定集数、CC 图标查看已有外挂字幕。
- 批量上传会弹出拖拽上传卡片，自动匹配字幕、目标视频和最终改名结果；生成预览后隐藏上传文件列表，锁定集数会被跳过。
- 支持在线字幕搜索：剧集可搜索本季/选中集字幕包，单集可点放大镜搜索；自动来源为 SubHD、Zimuku、射手网(伪) API 和 OpenSubtitles API。
- 在线字幕下载后会进入同一个匹配预览流程，确认语言后缀、集数和文件名后再写入，不会直接落盘。
- 自动搜索失败时，弹窗右侧仍提供 SubHD、Zimuku、射手网(伪)、OpenSubtitles 的手动搜索链接，可下载后回到本页手动上传。
- OpenSubtitles 默认搜索中文、英文、日文等多语言字幕；英文字幕结果可下载后提交给 `AI字幕生成(联动版)` 执行翻译。
- 可联动 `AI字幕生成(联动版)`：在整季工具栏或单集行点击 AI 图标提交生成任务，并在字幕匹配页面查看排队、生成、完成、忽略或失败状态。
- 在线搜索默认不使用 MoviePilot 系统代理，由容器自身网络出口决定；如需代理可在插件设置中手动开启。
- 插件设置可维护字幕站点根地址，以及射手网(伪) / OpenSubtitles API 地址和 API Key，域名、反代地址或 API 入口变化时无需更新插件版本。
- 匹配预览支持批量修改语言后缀，单条字幕仍可独立修正。
- 可选写入前繁体转简体：开启后仅处理中文后缀字幕，使用 Tongwen 词组和单字字典离线转换。
- 可选监听 MoviePilot 入库完成事件自动搜索字幕：默认关闭；仅在目标无外挂字幕、单个高置信结果且能明确匹配时自动写入。
- 支持批量清空选中集的外挂字幕，仅删除与视频同名开头的外挂字幕文件。
- `.strm` 会按本地视频目标参与统计和匹配。
- 上传字幕文件：`.srt`、`.ass`、`.ssa`、`.sbv`、`.sub`、`.vtt`、`.webvtt`。
- 上传 `.zip`、`.rar`、`.7z` 压缩包并自动解包其中的字幕文件。
- 新增轻量 Python RAR 依赖 `rarfile` 声明；上传 `.rar` 或 `.7z` 压缩包时默认使用容器内 `unar` / `lsar` 解包，缺失时会明确提示。
- 上传弹窗和插件设置会直接显示当前解压器状态；如缺失，可选择容器内临时安装 `unar`，或宿主机映射 `unar` 到容器。
- 插件设置可选择 RAR / 7Z 解压器处理方式：不处理、加载插件时尝试容器内安装、使用宿主机映射文件。
- 写入前可选“智能调轴”：优先用视频内置文本字幕做基准，没有内置字幕时用 `ffmpeg` 抽取音频并通过 FFT 互相关计算整体偏移；该功能可能占用 CPU 并造成短暂卡顿。
- 按目标视频文件名生成外挂字幕名并直接落盘。

命名策略：

- 默认生成 `视频文件名.语言标识.字幕后缀`。
- 语言标识使用媒体服务器常用的 ISO 639 三字母码，例如中文 `chi`、英文 `eng`、日文 `jpn`、韩文 `kor`。
- 例如：`Episode.Name.S01E01.chi.ass`。

限制说明：

- 当前版本主要支持可从 MoviePilot 本地媒体库读取到的本地视频文件；CD2、WebDAV、SMB、网盘挂载等远程路径暂不保证完整可用性。
- “信任整理历史路径”只用于减少慢路径批量探测，开启后插件会信任 MoviePilot 整理历史中的路径，不逐条访问目标文件；已有外挂字幕、内置字幕等状态可能显示不完整，写入字幕仍受挂载稳定性影响。
- SubHD 默认地址为 `https://subhd.tv`，优先使用豆瓣 ID 自动搜索，没有 ID 时使用标题关键词搜索，站点波动时可手动跳转。
- Zimuku 默认地址为 `https://zmk.pw`，使用标题关键词搜索候选字幕页，站点波动时可手动跳转。
- 射手网(伪) 配置 API Key 后访问 `https://api.assrt.net` 官方接口；未配置时不参与自动搜索。
- OpenSubtitles 配置 API Key 后访问 `https://api.opensubtitles.com/api/v1` 搜索中文、英文、日文字幕；下载时由插件使用 OpenSubtitles 用户名和密码后台登录换取 token。
- 自定义站点地址只填写根地址，例如 `https://subhd.tv` 或反代入口；插件会按各站当前路径拼接搜索页。
- `rarfile` 是最轻量的 Python RAR 封装层，但不是纯 Python 解压器；RAR5/压缩内容默认交给 `unar` 处理。
- 临时测试可在插件设置中选择“加载插件时尝试容器内安装”，或进入 MoviePilot 容器安装 `unar`；容器重建后可能失效。
- 长期建议直接使用 MoviePilot 容器内置的 `/usr/bin/unar`；如果容器没有内置，可把宿主机 `unar` 映射到容器内 `/usr/bin/unar`，并确保同目录或 PATH 中可找到 `lsar`。
- 智能调轴依赖容器内 `ffmpeg`、`ffprobe`、`numpy`、`pysubs2`；插件会通过 `requirements.txt` 声明 `pysubs2`，缺失时页面会禁用调轴开关。
- 当前版本会去掉同目录里已有字幕文件名中的 `.default` / `.forced` 标记，但不会自动新增这些标记。

如果需要宿主机映射，按实际路径给 MoviePilot 服务增加映射，例如：

```yaml
volumes:
  - /usr/bin/unar:/usr/bin/unar:ro
  - /usr/bin/lsar:/usr/bin/lsar:ro
```
