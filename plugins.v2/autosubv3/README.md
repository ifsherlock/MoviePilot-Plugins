# AI字幕生成(联动版) (AutoSubv3)

MoviePilot V2 插件，用于从视频音轨、内嵌字幕或外挂字幕中生成字幕，并可调用大模型翻译为中文。基于 autosub 系列插件继续整理，感谢原作者与后续贡献者。

## v3.5.57 更新

- 优化设置页布局，将 API 配置前移到基础设置后，便于先完成大模型连接配置。
- “获取模型”和“测试模型”的结果改为在 API 配置区就近显示，不再跳到页面顶部。
- 设置页底部新增保存和关闭操作，长表单编辑完成后无需回到顶部保存。

## 功能特性

- 支持 OpenAI API 及兼容接口（硅基流动、DeepSeek 等）
- 支持 faster-whisper 多种模型（tiny/base/small/medium/large-v3/large-v3-turbo）
- 入库自动生成字幕并用 AI 翻译为中文
- 支持双语字幕或纯中文字幕输出模式
- 支持批量翻译 + 并发处理，速度更快
- 支持中文视频检测并跳过翻译
- 支持无声音视频自动跳过
- 支持联动字幕匹配插件，从字幕匹配页面单集或整季提交 AI 字幕生成任务并查看状态

## 字幕提取策略

优先级：**外挂字幕 > 内嵌字幕 > Whisper 语音识别**

| 策略 | 说明 |
|------|------|
| 仅英文 | 仅使用英文字幕，无则用 ASR |
| 英文优先 | 优先英文，无则用其他语言，再无则用 ASR |
| 原音优先 | 优先原音字幕，无则用英文，再无则用 ASR |

## 推荐配置

### 翻译模型

推荐使用性价比高的无推理模型：
- `inclusionAI/Ling-flash-2.0` - 速度快，便宜
- `inclusionAI/Ling-mini-2.0` - 更便宜

### Whisper 模型

| 模型 | 速度 | 准确度 | 适用场景 |
|------|------|--------|----------|
| tiny | 最快 | 一般 | 快速预览 |
| base | 快 | 较好 | 日常使用 |
| small | 中等 | 好 | 质量优先 |
| medium | 慢 | 很好 | 高质量 |
| large-v3 | 最慢 | 最好 | 极致质量 |
| large-v3-turbo | 较快 | 很好 | 平衡之选 |

## 注意事项

1. 首次使用 Whisper 会自动下载模型（约 1GB），建议开启代理
2. 批量翻译可大幅提升速度，但过大会影响翻译质量
3. 中文视频检测到后会自动跳过翻译（可关闭）
4. 无声音视频会自动跳过并记录
5. 任务队列机制保证并发安全

## 常见问题

**Q: 翻译结果为空或失败？**
A: 检查 API Key 是否正确，查看日志确认错误原因

**Q: Whisper 提取很慢？**
A: 使用更小的模型（tiny/base），或考虑 GPU 加速

**Q: 批量翻译失败率高？**
A: 降低 batch_size（建议 10-20），检查模型能力
